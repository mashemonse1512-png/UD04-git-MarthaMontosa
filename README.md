## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Git instalado
- Node (opcional)
- Npm (opcional)
- Editor de texto (VSCode)

### Install

1. Clone the repo
   ```sh
   git clone https://github.com/mashemonse1512-png/UD04-git-MarthaMontosa
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   ```sh
   yarn install
   ```
